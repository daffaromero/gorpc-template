# gorpc-template

This repo holds the template for a gRPC, event-driven, Go backend.

## What it is

 - Event-driven
- gRPC
- PostgreSQL

## What it isn't

- A full-fledged example
- A tutorial

## Who it is for

- Mostly myself, but also you and anyone else who wants to learn more about gRPC, event-driven architecture, and Go.
